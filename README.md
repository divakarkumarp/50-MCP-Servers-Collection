## üìã What is the Model Context Protocol (MCP)?

The Model Context Protocol (MCP), introduced by Anthropic in November 2024, establishes a standardized, secure interface for AI models to interact with external tools‚Äîcode repositories, databases, files, web services, and more‚Äîvia a JSON-RPC 2.0-based protocol. MCP is already supported by Claude, Gemini, and OpenAI, and is rapidly being adopted by platforms like Replit, Sourcegraph, and Vertex AI.
